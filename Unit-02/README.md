# Unit 02 - Collections and Functions

Unit 02 covers Python collections (lists, tuples, sets, dictionaries) and
function design for modular programming.

## Lectures
- Lecture 01: Lists Fundamentals and Comprehensions
- Lecture 02: Tuples and Immutability
- Lecture 03: Sets and Dictionaries
- Lecture 04: Functions Basics and Parameters
- Lecture 05: Advanced Functions and Functional Patterns

## How to Use
- Open `slides.pdf` for in-class teaching.
- Use `notes.pdf` for detailed explanations and teaching flow.
- Run demo scripts in `demo/` during the lecture.

## Compile
From repo root:
```powershell
.\build.ps1
```
