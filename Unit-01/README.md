# Unit 01 - Introduction to Python

Unit 01 covers the foundations required to start programming in Python:
running Python, syntax, data types, strings, operators, and control flow.

## Lectures
- Lecture 01: Introduction and Working with Python
- Lecture 02: Basic Syntax, Comments, Dynamic Typing, Mutability
- Lecture 03: Tokens, Naming, Strings, and Numeric Types
- Lecture 04: Input/Output, Escape Sequences, Operators, Precedence
- Lecture 05: Decision Making and Looping Structures

Each lecture folder contains:
- `latex/` (LaTeX sources + aux files)
- `slides.pdf` (compiled slide deck)
- `notes.pdf` (compiled lecture notes)
- `demo/` (Python demo scripts referenced in slides/notes)
