# Unit 04 - GUI Programming and Data Connectivity

Unit 04 introduces building desktop GUI applications using Tkinter and storing
data using databases.

Syllabus coverage:
- Tkinter fundamentals and widgets
- Layout managers, menus, dialogs
- Event handling and input validation
- Database connectivity basics (Relational vs NoSQL)
- Python DB-API (CRUD), transactions, and error handling

## Lectures
- Lecture 01: Tkinter Introduction and Widgets
- Lecture 02: Layout Managers + Menus + Dialogs
- Lecture 03: Event Handling and Input Validation
- Lecture 04: Tkinter + Databases (Concepts and Integration)
- Lecture 05: DB-API 2.0 and CRUD (SQLite) + MongoDB Overview
- Lecture 06: Transactions and Database Error Handling

## How to Use
- Open `slides.pdf` for in-class teaching.
- Use `notes.pdf` for detailed explanations and teaching flow.
- Run demo scripts in `demo/` during the lecture.
