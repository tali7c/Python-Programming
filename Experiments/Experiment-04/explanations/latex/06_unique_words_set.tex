\section*{Objective}
Count the number of unique words in a sentence using sets.

\section*{Input}
A sentence.

\section*{Output}
Unique word count (and optionally the set of unique words).

\section*{Approach}
\begin{itemize}
  \item Split sentence into words.
  \item Normalize words (lowercase, remove punctuation if needed).
  \item Insert into a set to remove duplicates automatically.
\end{itemize}

\section*{Time Complexity}
\texttt{O(n)} average-case, where $n$ is the number of words/characters.
