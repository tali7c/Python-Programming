\section*{Objective}
Input a sentence and print each word in a separate line.

\section*{Input}
A sentence (string).

\section*{Output}
One word per line.

\section*{Approach}
Use \texttt{split()} to get the list of words, then print each word using a loop.

\section*{Time Complexity}
\texttt{O(n)} where $n$ is the number of characters (or words).
