\section*{Objective}
Count total number of vowels in a given string (case-insensitive).

\section*{Input}
A string.

\section*{Output}
Total vowel count.

\section*{Approach}
Convert the string to lowercase and count characters in \texttt{\{a,e,i,o,u\}}.

\section*{Time Complexity}
\texttt{O(n)}.
