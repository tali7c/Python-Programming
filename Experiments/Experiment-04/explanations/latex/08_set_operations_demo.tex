\section*{Objective}
Take two sets and apply various set operations.

\section*{Input}
Two sets (use the sample sets given in the lab manual):
\begin{itemize}
  \item \texttt{S1 = \{Red, yellow, orange, blue\}}
  \item \texttt{S2 = \{violet, blue, purple\}}
\end{itemize}

\section*{Output}
Union, intersection, difference, and symmetric difference.

\section*{Approach}
Use operators:
\begin{itemize}
  \item \texttt{S1 | S2} (union)
  \item \texttt{S1 \& S2} (intersection)
  \item \texttt{S1 - S2}, \texttt{S2 - S1} (difference)
  \item \texttt{S1 \string^ S2} (symmetric difference)
\end{itemize}

\section*{Time Complexity}
\texttt{O(n)} average-case for set operations.
