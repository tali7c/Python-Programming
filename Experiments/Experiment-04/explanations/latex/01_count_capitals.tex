\section*{Objective}
Count and display the number of capital (uppercase) letters in a given string.

\section*{Input}
A string.

\section*{Output}
Number of uppercase letters.

\section*{Approach}
Traverse each character and count when \texttt{ch.isupper()} is true.

\section*{Time Complexity}
\texttt{O(n)} where $n$ is the length of the string.
