# Experiment 01 - Python Installation and Basic Python Statements

## Objective
Get comfortable with:
- installing/running Python,
- basic statements and variables,
- numeric calculations using operators,
- and simple input/output.

## Prerequisites
- None (this is the first lab)

## Exercises

### Non-coding (setup)
- Installation + modes notes: `explanations/00_installation_and_modes.pdf`

### Programs (Scripts)
1. `scripts/01_age_type.py` - Store age and print its type
2. `scripts/02_hello_x.py` - Declare string variable `x = "Hello"` and print it
3. `scripts/03_print_data_types.py` - Print values of different data types
4. `scripts/04_basic_arithmetic_xyz.py` - Arithmetic on `x=9`, `y=7` (add/sub/mul/div)
5. `scripts/05_hypotenuse_pythagoras.py` - Hypotenuse using Pythagoras theorem
6. `scripts/06_simple_interest.py` - Simple interest calculation
7. `scripts/07_triangle_area_heron.py` - Triangle area from sides (Heron's formula + validation)
8. `scripts/08_seconds_converter.py` - Convert seconds to hours/minutes/seconds
9. `scripts/09_swap_without_temp.py` - Swap two numbers without extra variable
10. `scripts/10_sum_n_naturals.py` - Sum of first `n` natural numbers
11. `scripts/11_bitwise_truth_table.py` - Truth table for `&`, `|`, `^`
12. `scripts/12_bit_shifts.py` - Left shift and right shift
13. `scripts/13_membership_sequence.py` - Membership operator check in `(10,20,56,78,89)`
