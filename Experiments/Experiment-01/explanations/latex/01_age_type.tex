\section*{Objective}
Create a variable to store age and print its type using \texttt{type()}.

\section*{Input}
An integer age.

\section*{Output}
The age value and its type.

\section*{Approach}
Read age as an integer using \texttt{int(input(...))}, then print \texttt{type(age)}.

\section*{Time Complexity}
\texttt{O(1)}.
