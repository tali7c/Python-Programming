\section*{Objective}
Use the membership operator \texttt{in} to check whether a number exists in a given sequence.

\section*{Input}
An integer \texttt{n}.

\section*{Output}
Print whether \texttt{n} is present in the sequence \texttt{(10, 20, 56, 78, 89)}.

\section*{Approach}
Store the sequence in a tuple and use:
\begin{verbatim}
n in seq
\end{verbatim}

\section*{Time Complexity}
\texttt{O(m)} where $m$ is the length of the sequence (here $m=5$).
