\section*{Objective}
Swap two values without using a third (temporary) variable.

\section*{Input}
Two values (numbers or strings).

\section*{Output}
Values before and after swap.

\section*{Approach}
Use Python multiple assignment:
\begin{verbatim}
a, b = b, a
\end{verbatim}

\section*{Time Complexity}
\texttt{O(1)}.
