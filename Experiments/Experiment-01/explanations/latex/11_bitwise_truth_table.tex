\section*{Objective}
Print the truth table for bitwise operators \texttt{\&}, \texttt{|}, and \texttt{\string^}
for inputs $a,b \in \{0,1\}$.

\section*{Input}
No input (use $a,b \in \{0,1\}$).

\section*{Output}
A table showing \texttt{a\&b}, \texttt{a|b}, and \texttt{a\string^b}.

\section*{Approach}
Use nested loops over values \texttt{0} and \texttt{1} and print results.

\section*{Time Complexity}
\texttt{O(1)} (fixed small table).
