\section*{Objective}
Declare a string variable \texttt{x} and assign the value \texttt{"Hello"}.
Print the value of \texttt{x}.

\section*{Input}
No input.

\section*{Output}
Print \texttt{x}.

\section*{Approach}
Assign \texttt{x = "Hello"} and print it using \texttt{print(x)}.

\section*{Time Complexity}
\texttt{O(1)}.
