\section*{Objective}
Create values of different data types and print both the value and its type.

\section*{Input}
No input (use sample values in the program).

\section*{Output}
Value and type for each sample variable.

\section*{Approach}
Create variables of types like \texttt{int}, \texttt{float}, \texttt{bool}, \texttt{str},
\texttt{list}. Print each value and \texttt{type(value)}.

\section*{Time Complexity}
\texttt{O(k)} where $k$ is the number of sample values (constant in this exercise).
