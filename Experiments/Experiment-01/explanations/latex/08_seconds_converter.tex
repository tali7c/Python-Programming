\section*{Objective}
Convert a given number of seconds into hours, minutes, and remaining seconds.

\section*{Input}
A non-negative integer \texttt{total\_seconds}.

\section*{Output}
\texttt{hours}, \texttt{minutes}, and \texttt{seconds}.

\section*{Approach}
Use integer division and modulo:
\begin{itemize}
  \item \texttt{hours = total\_seconds // 3600}
  \item \texttt{rem = total\_seconds \% 3600}
  \item \texttt{minutes = rem // 60}
  \item \texttt{seconds = rem \% 60}
\end{itemize}

\section*{Time Complexity}
\texttt{O(1)}.
