\section*{Objective}
Compute left shift and right shift values of a given integer using \texttt{<<} and \texttt{>>}.

\section*{Input}
An integer \texttt{n} and a non-negative shift amount \texttt{k}.

\section*{Output}
Values of \texttt{n << k} and \texttt{n >> k}.

\section*{Approach}
Read \texttt{n} and \texttt{k}, then compute shifts directly using operators.

\section*{Time Complexity}
\texttt{O(1)}.
