\section*{Objective}
Check whether a given number is divisible by both 3 and 5.

\section*{Input}
An integer \texttt{n}.

\section*{Output}
Print whether \texttt{n} is divisible by both 3 and 5.

\section*{Approach}
Use the condition \texttt{n \% 3 == 0 and n \% 5 == 0}.

\section*{Time Complexity}
\texttt{O(1)}.
