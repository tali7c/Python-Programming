\section*{Objective}
Check whether a given number is a multiple of 5.

\section*{Input}
An integer \texttt{n}.

\section*{Output}
Print whether \texttt{n \% 5 == 0}.

\section*{Approach}
Compute \texttt{n \% 5}. If remainder is 0, it is a multiple of 5.

\section*{Time Complexity}
\texttt{O(1)}.
