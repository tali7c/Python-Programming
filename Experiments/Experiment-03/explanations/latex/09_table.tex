\section*{Objective}
Print the multiplication table for a given number.
\section*{Input}
An integer \texttt{n}.
\section*{Output}
Table from \texttt{n * 1} to \texttt{n * 10}.
\section*{Approach}
Use a loop from 1 to 10 and print each product.
\section*{Time Complexity}
\texttt{O(10)} which is constant.
