\section*{Objective}
Check whether a given integer is an Armstrong number.
\section*{Input}
An integer \texttt{n}.
\section*{Output}
Message indicating Armstrong or not.
\section*{Approach}
Count digits, raise each digit to that power, and compare with the original.
\section*{Time Complexity}
\texttt{O(d)} where \texttt{d} is the number of digits.
