\section*{Objective}
Check if a given integer is prime.
\section*{Input}
An integer \texttt{n}.
\section*{Output}
Message indicating prime or not.
\section*{Approach}
Test divisibility from 2 to \(\sqrt{n}\). If any divides, it is not prime.
\section*{Time Complexity}
\(\mathcal{O}(\sqrt{n})\).
