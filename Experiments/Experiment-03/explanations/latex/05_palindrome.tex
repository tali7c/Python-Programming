\section*{Objective}
Check whether an input string or number is a palindrome.
\section*{Input}
A string or number.
\section*{Output}
Message indicating palindrome or not.
\section*{Approach}
Reverse the input and compare with the original.
\section*{Time Complexity}
\texttt{O(n)} for length \texttt{n}.
