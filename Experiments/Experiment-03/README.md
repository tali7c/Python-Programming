# Experiment 03 - Loops (Practice Set)

## Objective
Practice writing Python programs using **loops** (for/while), basic conditionals,
and simple numeric/string logic.

## Prerequisites
- Variables, operators, type conversion
- `if/elif/else`
- `for` / `while` loops

## Exercises (Scripts)
1. `scripts/01_factorial.py` - Factorial of a number
2. `scripts/02_armstrong.py` - Armstrong number check
3. `scripts/03_fibonacci.py` - Fibonacci series (iterative)
4. `scripts/04_prime.py` - Prime number check
5. `scripts/05_palindrome.py` - Palindrome check (string/number)
6. `scripts/06_sum_digits.py` - Sum of digits
7. `scripts/07_divisible_5_7.py` - Numbers divisible by 5 or 7 (1..100)
8. `scripts/08_lower_to_upper.py` - Convert lowercase to uppercase
9. `scripts/09_table.py` - Multiplication table
10. `scripts/10_pattern.py` - Pattern printing
11. `scripts/11_harmonic_series.py` - Harmonic series sum

## Explanations (PDF)
- Compiled PDFs are in `explanations/`
- LaTeX sources are in `explanations/latex/`

## Evaluation Rubric (Suggested)
- Correctness (test cases): 4
- Code clarity (naming, structure): 2
- Input handling / edge cases: 2
- Output formatting: 1
- Viva / explanation: 1

## Viva Questions (Sample)
1. Difference between `for` and `while` loops?
2. What is an Armstrong number and why does the power equal the digit count?
3. Why do we check divisors only up to `sqrt(n)` for prime testing?
4. What is the difference between reversing a string and reversing a number?
5. What happens if `n = 0` for factorial and Fibonacci?
