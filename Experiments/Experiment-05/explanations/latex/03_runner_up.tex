\section*{Objective}
Find the second highest distinct score.
\section*{Input}
Number of scores and the score list.
\section*{Output}
Runner-up score.
\section*{Approach}
Remove duplicates with a set and pick the second largest.
\section*{Time Complexity}
\texttt{O(n log n)} due to sorting.
