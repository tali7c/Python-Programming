# Lab Experiments

These experiments provide hands-on practice aligned with the lecture sequence.

## Experiments in this repository
- Experiment 03: Loops and number/string programs
- Experiment 04: Strings and sets
- Experiment 05: Lists/tuples/dictionaries and mini-apps
- Experiment 06: Functions, recursion, and lambda

## Folder conventions
- `scripts/`: Python solutions for each exercise
- `explanations/`: compiled PDFs for each exercise
- `explanations/latex/`: LaTeX sources + LaTeX auxiliary files
