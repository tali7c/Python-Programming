\section*{Objective}
Create a dictionary using keys from one list and values from another.
\section*{Input}
Two lists of equal length.
\section*{Output}
A dictionary created from the lists.
\section*{Approach}
Use \texttt{zip} to pair keys and values, then convert to dict.
\section*{Time Complexity}
\texttt{O(n)}.
