\section*{Objective}
Generate Fibonacci series using recursion.
\section*{Input}
Number of terms \texttt{n}.
\section*{Output}
List of the first \texttt{n} Fibonacci numbers.
\section*{Approach}
Define \texttt{fib(n)} recursively and build the list.
\section*{Time Complexity}
\(\mathcal{O}(2^n)\) for naive recursion.
