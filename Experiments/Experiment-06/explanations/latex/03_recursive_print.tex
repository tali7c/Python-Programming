\section*{Objective}
Print numbers from 1 to \texttt{n} using recursion.
\section*{Input}
An integer \texttt{n}.
\section*{Output}
Numbers 1 through \texttt{n}.
\section*{Approach}
Recursive call with \texttt{n-1} then print \texttt{n}.
\section*{Time Complexity}
\texttt{O(n)}.
